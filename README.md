# Robot Animation

A simple webpage with moving robot animations.

## What it does

- Shows 4 colored robots on the screen
- Click any robot to make it walk across and back
- Robots stop when they return to where they started

## How to use

1. Open index.html in your web browser
2. Click on a robot to make it move
3. Each robot moves independently

## Files

- index.html - Main webpage
- css/style.css - Robot styles and animations  
- js/alien.js - Robot movement code
